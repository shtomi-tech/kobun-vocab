### id 259 たてまつる（奉る／動）— 差し上げる（謙・与ふ）／（お）〜申し上げる（謙・補助）／お乗りになる（尊・乗る）／お召しになる（尊・着る）／召し上がる（尊・食ふ／飲む）
- POINT: 謙譲の本動詞・補助動詞に加え、「乗る・着る・食ふ」を婉曲的に「お〜になる」と表す尊敬の用法もある特殊な語。文脈で謙譲か尊敬か判断する。
- 例文: 御文をたてまつる。
- 訳: お手紙を差し上げる。

### id 260 さぶらふ／さうらふ（候ふ／動）— お仕えする、お控えする（謙・仕ふ／あり）／参上する（謙・行く／来）／あります、ございます（丁・あり）／〜です、〜ます（丁・補助）
- POINT: 謙譲（お仕えする）と丁寧（あります・です）の両方の用法を持つ。主語が誰かに仕えている（謙譲）か、単に丁寧に述べている（丁寧）かで判別する。
- 例文: 御前にさぶらふ。
- 訳: 御前にお控え申し上げる。

### id 261 はべり（侍り／動）— お仕えする、お控えする（謙・仕ふ／あり）／あります、ございます（丁・あり）／〜です、〜ます（丁・補助）
- POINT: id260「さぶらふ」とほぼ同じ用法（謙譲・丁寧）を持つ類語。平安中期以降「さぶらふ」に代わって多用される。
- 例文: 心得はべり。
- 訳: 承知いたします（丁寧の「〜ます」）。

### id 262 まゐらす（参らす／動）— 差し上げる、献上する（謙・与ふ）／（お）〜申し上げる、〜して差し上げる（謙・補助）
- POINT: 謙譲の本動詞・補助動詞。id263「まゐる」と語根が近いが、こちらは「差し上げる」の意に特化する。
- 例文: 御文をまゐらす。
- 訳: お手紙を差し上げる。

### id 263 まゐる（参る／動）— 参上する（謙・行く／来）／差し上げる、献上する（謙・与ふ）／し申し上げる、して差し上げる（謙・す）／召し上がる（尊・食ふ／飲む）
- POINT: 謙譲の本動詞。「参上する」と「差し上げる」の両方の意味を持つ点に注意。まれに「食ふ・飲む」の尊敬（召し上がる）にもなる特殊用法がある。
- 例文: 御前にまゐる。
- 訳: 御前に参上する。

### id 264 きこしめす（聞こし召す／動）— お聞きになる（尊・聞く）／召し上がる（尊・食ふ／飲む）／お治めになる（尊・治む）
- POINT: 「聞く」の尊敬語だが、「食ふ・飲む」「治む」の尊敬語としても使われる多義の敬語。文脈判断が必須。id283「しろしめす」と語形が似ているため区別する。
- 例文: 御琴の音をきこしめす。
- 訳: お琴の音をお聞きになる。

### id 265 つかうまつる／つかまつる（仕うまつる／仕る／動）— お仕えする（謙・仕ふ）／し申し上げる、して差し上げる（謙・す）／（お）〜申し上げる（謙・補助）
- POINT: 「仕ふ」の謙譲語に加え、「す」の謙譲語（し申し上げる）としても使われる。
- 例文: 御用をつかうまつる。
- 訳: ご用をお仕え申し上げる（して差し上げる）。

### id 266 たまふ（給ふ・賜ふ／動）— お与えになる（尊・与ふ）／〜なさる、お〜になる（尊・補助）／〜（ており）ます、〜させていただく（謙・補助）
- POINT: 四段活用（尊敬：お与えになる／〜なさる）と下二段活用（謙譲：〜させていただく）で敬意の方向が反転する最重要語。活用形の識別が入試最頻出。
- 例文: 帝、御衣をたまふ。（四段・尊敬）
- 訳: 帝が、お召し物をお与えになる。

### id 267 たぶ／たうぶ（給ぶ・賜ぶ／動）— お与えになる（尊・与ふ）／〜なさる、お〜になる（尊・補助）
- POINT: id266「たまふ」の音変化した類語。用法・意味とも「たまふ」の尊敬用法に対応する。
- 例文: 御言葉をたびたまへ。
- 訳: お言葉をお与えください。

### id 268 いざたまへ／いざさせたまへ／いざたべ（連）— さあいらっしゃい
- POINT: 「いざ（さあ）」＋id266・267「たまへ／たべ」の命令形が結びついた誘いかけの決まり文句。
- 例文: いざたまへ、ともに参らむ。
- 訳: さあいらっしゃい、一緒に参りましょう。

### id 269 おはす（動）— いらっしゃる、おいでになる、おありになる（尊・あり／行く／来）／〜ていらっしゃる、〜なさる、お〜になる（尊・補助）
- POINT: 「あり・行く・来」の尊敬語（本動詞）、および尊敬の補助動詞として使われる。id270「おはします」より敬意がやや低い。
- 例文: 宮、こなたにおはす。
- 訳: 宮様が、こちらにいらっしゃる。

### id 270 おはします（動）— いらっしゃる、おいでになる、おありになる（尊・あり／行く／来）／〜ていらっしゃる、〜なさる、お〜になる（尊・補助）
- POINT: id269「おはす」に「〜します」がついた形で、敬意がより高い。意味・用法はid269と同じ。
- 例文: 帝、清涼殿におはします。
- 訳: 帝が、清涼殿にいらっしゃる。

### id 271 ます（動）— いらっしゃる、おいでになる、おありになる（尊・あり／行く／来）／〜ていらっしゃる、〜なさる、お〜になる（尊・補助）
- POINT: id269「おはす」と同系統の尊敬語（あり・行く・来）。用例はやや少なく、主に和歌や古い文体に見られる。
- 例文: 神ますところなり。
- 訳: 神がいらっしゃる場所である。

### id 272 まします（動）— いらっしゃる、おいでになる、おありになる（尊・あり）／〜ていらっしゃる、〜なさる、お〜になる（尊・補助）
- POINT: id271「ます」に「〜します」がついた形。id270「おはします」と同様、敬意がより高い形。ただし対象は「あり」のみ（行く・来の意味は持たない）。
- 例文: 御所にまします。
- 訳: 御所にいらっしゃる。

### id 273 います（動）— いらっしゃる、おいでになる、おありになる（尊・あり／行く／来）／〜ていらっしゃる、〜なさる、お〜になる（尊・補助）
- POINT: id269「おはす」・id271「ます」と同系統の尊敬語。「あり・行く・来」いずれの意味も持つ。
- 例文: 山にいます聖なり。
- 訳: 山にいらっしゃる聖である。

### id 274 いまそかり／いますがり（動）— いらっしゃる、おいでになる、おありになる（尊・あり）／〜ていらっしゃる、〜なさる、お〜になる（尊・補助）
- POINT: id273「います」に「あり」が付いた形の尊敬語。意味・用法はid269〜273と同系統。
- 例文: 都にいまそかりし人なり。
- 訳: 都にいらっしゃった人である。

### id 275 のたまふ（宣ふ／動）— おっしゃる（尊・言ふ）
- POINT: 「言ふ」の尊敬語。id276「のたまはす」より敬意がやや低い。
- 例文: 帝、のたまふやう。
- 訳: 帝がおっしゃることには。

### id 276 のたまはす（宣はす／動）— おっしゃる（尊・言ふ）
- POINT: id275「のたまふ」に「す（使役・尊敬の助動詞）」がついた形で、敬意がより高い。
- 例文: 后、のたまはす。
- 訳: 后がおっしゃる。

### id 277 おほす（仰す／動）— おっしゃる（尊・言ふ）
- POINT: 「言ふ」の尊敬語。id275・276と同系統だが、より公的・命令的な発言（詔・宣旨など）に使われることが多い。
- 例文: 帝、おほせらる。
- 訳: 帝が、仰せになる。

### id 278 きこゆ（聞こゆ／動）— 申し上げる（謙・言ふ）／（お）〜申し上げる（謙・補助）
- POINT: 「言ふ」の謙譲語（本動詞・補助動詞）。「聞こえる（自然と）」の意味もあるが、敬語の文脈では「申し上げる」と訳す。
- 例文: 御礼をきこゆ。
- 訳: お礼を申し上げる。

### id 279 きこえさす（聞こえさす／動）— 申し上げる（謙・言ふ）／（お）〜申し上げる（謙・補助）
- POINT: id278「きこゆ」に「さす（使役・尊敬の助動詞）」がついた形で、敬意がより高い。用法はid278と同じ。
- 例文: 御文をきこえさす。
- 訳: お手紙を差し上げ申し上げる。

### id 280 まうす（申す／動）— 申し上げる（謙・言ふ）
- POINT: 「言ふ」の謙譲語。id278「きこゆ」と類義で、現代語「申す」にそのまま繋がる。
- 例文: 御礼をまうす。
- 訳: お礼を申し上げる。

### id 281 そうす（奏す／動）— （天皇・上皇・法皇に）申し上げる（謙・言ふ）
- POINT: 「言ふ」の謙譲語だが、対象が天皇・上皇・法皇に限定される最高敬語。id282「けいす」と対で覚える。
- 例文: 天皇にそうす。
- 訳: 天皇に申し上げる。

### id 282 けいす（啓す／動）— （皇后・中宮・東宮などに）申し上げる（謙・言ふ）
- POINT: id281「そうす」と対の謙譲語。対象が皇后・中宮・東宮に限定される点で区別する。
- 例文: 中宮にけいす。
- 訳: 中宮に申し上げる。

### id 283 しろしめす（知ろし召す／動）— ご存じである（尊・知る）／お治めになる（尊・領る）
- POINT: 「知る」の尊敬語（ご存じである）と「領る（治める）」の尊敬語（お治めになる）の両方を持つ。id264「きこしめす」と語形が似ているため区別に注意。
- 例文: 帝、天下をしろしめす。
- 訳: 帝が、天下をお治めになる。

### id 284 ぎゃうかう／みゆき（行幸／名）— （天皇の）お出まし
- POINT: 天皇の外出を指す最高敬語の名詞。id285「御幸（上皇等）」・id286「行啓（皇后等）」と対象の身分で使い分ける。
- 例文: 天皇のぎゃうかうあり。
- 訳: 天皇のお出ましがある。

### id 285 ごかう／みゆき（御幸／名）— （上皇・法皇・女院の）お出まし
- POINT: id284「行幸」の対象違い版。上皇・法皇・女院の外出を指す。
- 例文: 上皇のごかうを仰ぐ。
- 訳: 上皇のお出ましを拝する。

### id 286 ぎゃうけい（行啓／名）— （皇后・中宮・東宮などの）お出まし
- POINT: id284・285と対応する語。皇后・中宮・東宮の外出を指す。
- 例文: 中宮のぎゃうけいを迎ふ。
- 訳: 中宮のお出ましをお迎えする。
