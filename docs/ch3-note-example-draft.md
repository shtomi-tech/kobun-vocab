### id 129 ねんず（念ず／動）— 祈る、祈念する／我慢する、こらえる
- POINT: 「祈る」が原義。そこから「心の中で祈るように我慢する＝こらえる」の意に転じた。
- 例文: 心の中にねんじて、涙をこらふ。
- 訳: 心の中でじっと祈るように我慢して、涙をこらえる。

### id 130 しのぶ（忍ぶ／動）— 我慢する、こらえる／人目を避ける、ひそかに行う
- POINT: 「我慢する」と「人目を避ける」の二方向。id131「偲ぶ」とは漢字も意味も別語なので注意。
- 例文: 人目をしのびて、通ふ。

### id 131 しのぶ（偲ぶ／動）— 思い慕う、懐かしく思う
- POINT: id130「忍ぶ」と同音異義語。「懐かしく思い出す」の意味は漢字「偲ぶ」の方。

- 訳: 亡くなった人を懐かしく思って、涙する。

### id 132 いかで（副）— どうして（〜推量）／どのようにして（〜推量）／どうにかして（〜願望）
- POINT: 疑問・反語の推量（「どうして〜か」）と、願望（「どうにかして〜したい」）の2方向。下に来る語（む・ばや等）で判別。
- 例文: いかでこの事を知らむ。
- 訳: どうにかしてこのことを知りたい。

### id 133 いかにして（連）— どうして（〜推量）／どのようにして（〜推量）／どうにかして（〜願望）
- POINT: id132「いかで」とほぼ同義の類語。意味・用法とも対応する。
- 例文: いかにしてかの人に逢はむ。
- 訳: どうにかしてあの人に会いたい。

### id 134 いかに（副）— どのように、どうなのか／どうして／どれほど

### id 183 ほど（程／名）— 程度、段階、様子、具合／時、頃、あいだ／広さ、距離、途中、場所、辺り、長さ、高さ、深さ／身分、年齢
- POINT: 「時間」「空間」「程度」「身分」のどれを指すか、文脈判断が必須の多義語。
- 例文: 三年ばかりのほどに、世を去りぬ。
- 訳: 3年ほどの間に、この世を去った。

### id 184 さうなし（左右無し／形）— あれこれためらわない、言うまでもない、容易だ
- POINT: 「左右（あれこれ思案すること）」が「無し」＝迷わない、が原義。id185・186（双無し・二無し＝比べるものがない）とは別語源なので混同注意。
- 例文: さうなく引き受けたり。
- 訳: あれこれためらわずに（すぐに）引き受けた。

### id 185 さうなし（双無し／形）— またとない、並ぶものがない
- POINT: id184「左右無し」と同音異義語。「双（ふたつと並ぶものがない）」＝「またとない」の意。id186「になし」と同義。
- 例文: 天下にさうなき美人なり。
- 訳: この世に並ぶ者がない美人である。

### id 186 になし（似無し・二無し／形）— またとない、並ぶものがない
- POINT: id185「さうなし（双無し）」とほぼ同義の類語。「二つとない＝比類ない」が原義。
- 例文: になき宝を得たり。
- 訳: 比類ない宝を手に入れた。
