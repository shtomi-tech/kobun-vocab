### id 241 ぐす（具す／動）— 備わる／備える／付いていく、付き従う、連れ添う／連れていく、付き従える／添える／携帯する
- POINT: 自動詞（備わる）と他動詞（備える、連れて行く）の両方で使われる。「〜を具して」で「〜を連れて」の意になる頻出表現。
- 例文: 供の者を具して出づ。
- 訳: 供の者を連れて出かける。

### id 242 な（名／名）— うわさ、評判、名声
- POINT: 現代語「名（名前）」ではなく、古語では「うわさ、評判」の意味が中心。id243・244の複合語の元になる。
- 例文: 世に名の高き人なり。
- 訳: 世間で評判（うわさ）の高い人である。

### id 243 うきな（憂き名／名）— つらいうわさ、悪い評判
- POINT: id242「な」の複合語。「憂き（つらい）」＋「名（うわさ）」で「悪い評判」の意。
- 例文: うきなを立てられて嘆く。
- 訳: つらいうわさを立てられて嘆く。

### id 244 なきな（無き名／連）— 何の事実もないうわさ、根拠のないうわさ、身に覚えのないうわさ
- POINT: id242「な」の複合語。「無き（事実がない）」＋「名（うわさ）」で「根拠のないうわさ」の意。
- 例文: なきなを負ひて苦しむ。
- 訳: 身に覚えのないうわさを背負って苦しむ。

### id 245 こころにくし（心憎し／形）— 奥ゆかしい、心ひかれる、すばらしい
- POINT: 「憎らしいほど」という程度の強さから、マイナスではなく「奥ゆかしい、すばらしい」というプラスの評価に転じた語。
- 例文: こころにくきふるまひをする人なり。
- 訳: 奥ゆかしい振る舞いをする人だ。

### id 246 にくし（憎し／形）— しゃくだ、憎らしい／みっともない、見苦しい／感心だ、あっぱれだ
- POINT: id245「こころにくし」とは反対に、「しゃくだ」というマイナス評価に加え、「感心だ、あっぱれだ」という反語的なプラス評価も持つ点に注意。
- 例文: にくきほどに気の利きたる返事なり。
- 訳: しゃくなほど（感心するほど）気の利いた返事である。

### id 247 にくからず（憎からず／連）— 好ましい、奥ゆかしい
- POINT: id246「にくし」の打消形。「憎くはない＝好ましい」という婉曲的な好意の表現。
- 例文: にくからぬ人と契りを結ぶ。
- 訳: 好ましい人と約束を結ぶ。

### id 248 あさまし（形）— 驚きあきれることだ、意外だ／（思いがけず）すばらしい／ひどい、情けない、見苦しい
- POINT: id255「あさむ」と同根。予想外の事態への驚きが原義で、良い意味にも悪い意味にも使われる。
- 例文: あさましきまでに変はり果てぬ。
- 訳: 驚きあきれるほどに変わり果ててしまった。

### id 249 ひがこと（僻事／名）— 間違い、噓、道理に合わないこと、不都合なこと
- POINT: 「僻（ひが＝ゆがんだ）」＋「事」で、id250〜253「ひが〜」シリーズの元になる語。「間違い、道理に合わないこと」の意。
- 例文: ひがことを申して、恥をかく。
- 訳: 間違ったことを申し上げて、恥をかく。

### id 250 ひがめ（僻目／名）— 見間違い
- POINT: id249「ひがこと」の複合語。「目」の見間違いを指す。
- 例文: ひがめにや、人影の見ゆる。
- 訳: 見間違いだろうか、人影が見える。

### id 251 ひがみみ（僻耳／名）— 聞き間違い
- POINT: id250「ひがめ」と対応する「耳」の聞き間違い。
- 例文: ひがみみにや、声の聞こゆる。
- 訳: 聞き間違いだろうか、声が聞こえる。

### id 252 ひがおぼえ（僻覚え／名）— 記憶違い
- POINT: id249「ひがこと」の複合語。「覚え（記憶）」の間違いを指す。
- 例文: ひがおぼえにて、日を誤る。
- 訳: 記憶違いで、日にちを間違える。

### id 253 ひがひがし（僻僻し／形）— ひねくれている、まともでない
- POINT: id249「ひがこと」の形容詞形。「ひねくれている、まともでない」性格を表す。
- 例文: ひがひがしき心のみ多かる人なり。
- 訳: ひねくれた心ばかり多い人だ。

### id 254 あきる（呆る／動）— どうしてよいかわからなくなる、呆然とする、途方にくれる
- POINT: 現代語「飽きる」ではなく、「呆然とする、途方にくれる」の意。漢字「呆」で区別する。
- 例文: 事の意外さにあきれぬ。
- 訳: 事の意外さに呆然としてしまった。

### id 255 あさむ（動）— 驚きあきれる、呆気にとられる
- POINT: id248「あさまし」の動詞形。「驚きあきれる」の意。
- 例文: あさみて物も言はず。
- 訳: 驚きあきれて、ものも言えない。

### id 256 ほる（惚る／動）— 呆然とする、ぼんやりする／耄碌する、年老いてにぶくなる
- POINT: 「惚ける（ぼける）」の由来。「呆然とする」から「年老いて耄碌する」まで意味が広がる。
- 例文: 年老いて、ややほれゆく。
- 訳: 年をとって、次第にぼんやりしていく（耄碌していく）。

### id 257 ほく（惚く・呆く／動）— ぼんやりする、ほうける
- POINT: id256「ほる」の類語。同じく「ぼんやりする、ほうける」の意。
- 例文: 心ほけて、ものを忘る。
- 訳: 心がぼんやりして、物事を忘れる。

### id 258 まどふ（惑ふ／動）— 途方にくれる、戸惑う、迷う／心が乱れる、思い悩む／うろたえる、慌てる／ひどく（〜する）（動詞の連用形について）
- POINT: 「途方にくれる」が基本だが、補助動詞「〜まどふ」で「ひどく〜する」という程度の強調にもなる点に注意。
- 例文: 道に迷ひて、まどふ。
- 訳: 道に迷って、途方にくれる。
