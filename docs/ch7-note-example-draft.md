### id 357 〜さす（〜止す／接尾）— 〜するのを途中でやめる／〜するのが途中でやんでいる
- POINT: 動詞の連用形について「途中でやめる」の意になる接尾語。id330「〜あへず」（できない）と混同しないよう区別する。
- 例文: 言ひさして、黙りぬ。
- 訳: 言いかけてやめて、黙ってしまった。

### id 358 よさり／ようさり（夜去り／名）— 夜（になる頃）
- POINT: 「去り（近づく）」で「夜になる頃」を表す。id359「ゆふさり」と同じ構造。
- 例文: よさりつかた、月出づ。
- 訳: 夜になる頃、月が出る。

### id 359 ゆふさり（夕去り／名）— 夕方（になる頃）
- POINT: id358「よさり」と同じ構造で「夕方になる頃」を表す。
- 例文: ゆふさりに、文をおこす。
- 訳: 夕方になる頃、手紙をよこす。

### id 360 よ（も）すがら（夜（も）すがら／副）— 一晩中、夜通し
- POINT: 「すがら（〜の間ずっと）」が「夜」につくと「一晩中」。id361「よひとよ」と類義。
- 例文: よもすがら、月を眺む。
- 訳: 一晩中、月を眺める。

### id 361 よひとよ（夜一夜／副）— 一晩中、夜通し
- POINT: 「夜」を「一夜（ひとよ）」丸ごと、で「一晩中」の意。id360「よすがら」の類語。
- 例文: よひとよ、寝られず。
- 訳: 一晩中、眠れない。

### id 362 ひぐらし（日暮らし／副）— 一日中、朝から晩まで
- POINT: 「日（一日）」を「暮らす」で「一日中」の意。虫の「ひぐらし（蜩）」と同音だが別語なので注意。
- 例文: ひぐらし、書を読みて過ぐす。
- 訳: 一日中、書物を読んで過ごす。

### id 363 ひねもす／ひめもす（終日／副）— 一日中、朝から晩まで
- POINT: id362「ひぐらし」と同義の類語。漢字「終日」からも意味を連想できる。
- 例文: ひねもす、雨降る。
- 訳: 一日中、雨が降る。

### id 364 ひひとひ（日一日／副）— 一日中、朝から晩まで
- POINT: id361「よひとよ」の昼版。「日」を「一日（ひとひ）」丸ごと、で「一日中」の意。
- 例文: ひひとひ、庭を眺めて過ぐす。
- 訳: 一日中、庭を眺めて過ごす。

### id 365 あかつき（暁／名）— 夜明け前、未明
- POINT: 「夜明け前、まだ暗いうち」を指し、現代語「暁（明け方全般）」よりも早い時間帯限定。男女の後朝（きぬぎぬ）の別れの場面で頻出。
- 例文: あかつきに、家路を急ぐ。
- 訳: 夜明け前に、家路を急ぐ。

### id 366 けさう（懸想／名）— 想いを懸けること、恋い慕うこと、恋慕

### id 367 けさうぶみ（懸想文／名）— 恋文

### id 368 おもひかく（思ひ懸く／動）— 思いを懸ける、恋い慕う／予期する、予想する

### id 370 うちつけなり（打ち付けなり／形動）— 突然だ、にわかだ／軽率だ、軽々しい／露骨だ、無遠慮だ
- POINT: 「打ち付け（ぶつかる）」が原義で、物事が急に起こる様子から「軽率だ」「露骨だ」まで意味が広がる。
- 例文: うちつけなる恋に心乱る。

### id 371 ゆくりなし（形）— 突然だ、にわかだ／軽率だ、不注意だ
- POINT: id370「うちつけなり」と類義。「突然」から「軽率だ」まで似た意味の広がりを持つ。
- 例文: ゆくりなく人の来たれば、驚く。

### id 372 にはかなり（俄なり／形動）— 突然だ、だしぬけだ
- POINT: id370・371と類義だが、「にはかなり」は単純に「急だ、だしぬけだ」の意味に絞られる。
- 例文: にはかに雨降り出づ。

### id 373 あるじ（主・饗／名）— 主人として客をもてなすこと、ごちそう、饗応

### id 375 なま〜（生〜／接頭）— 未熟な〜、若い〜、中途半端な〜
