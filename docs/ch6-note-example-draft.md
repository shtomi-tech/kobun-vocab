- POINT: id345の短縮形。意味・用法とも同じ。
- 例文: さもあれ、われはわが道を行かむ。
- 訳: どうとでもなれ、私は私の道を行こう。

### id 347 さはれ（感）— どうとでもなれ、なるようになれ、ままよ
- POINT: id345・346と同系列の感動詞。「まあどうにでもなれ」という開き直りの気持ちを表す。
- 例文: さはれ、今さら悔ゆまじ。
- 訳: ままよ、今さら後悔すまい。

### id 348 さかし（然かし／連）— そうだよ
- POINT: 「さ（そう）」＋「かし（念押しの終助詞）」で、相手の発言に同意する「そうだよ」の意。
- 例文: さかし、まことにさなり。
- 訳: そうだよ、本当にその通りだ。

### id 349 さこそいへ（然こそ言へ／連）— そうは言うものの
- POINT: 「さ（そう）」＋係助詞「こそ」＋「言へ（已然形・逆接）」で、「そうは言うものの」という逆接の慣用句。
- 例文: さこそいへ、心には従はず。
- 訳: そうは言うものの、心では納得しない。

### id 350 さしもの（然しもの／連）— あれほどの、さすがの
- POINT: 「さ（そう）」＋「しも（強意）」＋「の」で、「あれほどの、さすがの」という強調の連体表現。
- 例文: さしもの豪傑も、涙を流す。
- 訳: あれほどの（さすがの）豪傑も、涙を流す。

### id 351 さもあり（然もあり／連）— そのようだ、そうだ／当然のことだ、もっともなことだ
- POINT: 「さ（そう）」＋「もあり（である）」で、「そのようだ」から「当然だ、もっともだ」まで意味が広がる。
- 例文: 人の怒るも、さもあり。
- 訳: 人が怒るのも、当然のことだ（もっともだ）。

### id 352 さらでだに（然らでだに／連）— そうでなくてさえ
- POINT: 「さらで（そうでなくて）」＋「だに（〜さえ）」で、「ただでさえ」という強調の前置き表現。
- 例文: さらでだにわびしきに、雨さへ降る。
- 訳: そうでなくてさえつらいのに、雨まで降る。

### id 353 さること（然ること／連）— そのようなこと／当然のこと、もっともなこと
- POINT: 「さ（そう）」＋「こと」で、「そのようなこと」から「当然のこと」まで意味が広がる。
- 例文: さることは、世に多かり。
- 訳: そのようなことは、世間に多い。

### id 354 さるもの（然るもの／連）— そのようなもの／当然のこと、もっともなこと
- POINT: id353「さること」の類語。「もの」を対象とした表現。
- 例文: さるものと心得て、備ふ。
- 訳: 当然のことと心得て、備える。

### id 355 さればこそ（然ればこそ／連）— 思ったとおりだ、案の定だ
- POINT: 「さ（そう）」＋「あれば（已然形＋ば＝確定条件）」＋係助詞「こそ」で、「思った通りだ」という確信の表現。
- 例文: さればこそ、かく成りぬれ。
- 訳: 思ったとおりだ、このようになった。

### id 356 さればよ（然ればよ／連）— 思ったとおりだ、案の定だ
- POINT: id355「さればこそ」の類語。「よ」は詠嘆の終助詞。
- 例文: さればよ、あやまたじと思ひしに。
- 訳: 思ったとおりだ、間違えまいと思っていたのに。
