### id 7 あからさまなり（形動）— 一時的だ、ちょっとのあいだだ
- POINT: 現代語「あからさま（露骨）」とは全く別の意味。「ほんの少しの間」という時間の短さを表す。
- 例文: あからさまに立ち出でて、すなはち帰りぬ。
- 訳: ほんのちょっとの間外に出て、すぐに帰った。

### id 8 あからめ（あから目／名）— よそ見、わき見／浮気
- POINT: 「あから」は「あからさまなり」と同根で「ちょっとの間」。「目」がそれると考えると意味がつながる。
- 例文: あからめもせで、書を読む。
- 訳: よそ見もしないで、書物を読む。

### id 9 おどろく（驚く／動）— 驚く、びっくりする／はっと気づく／目が覚める
- POINT: 現代語の「驚く（びっくり）」の意味に加え、「はっと気づく」「眠りから覚める」の意味が古文では重要。
- 例文: 風の音に、秋来ぬとおどろかれぬる。
- 訳: 風の音で、秋が来たとはっと気づかされた。

### id 10 おどろかす（驚かす／動）— 驚かせる／気づかせる／目を覚まさせる、起こす
- POINT: 「おどろく」の他動詞形。3つの意味も「おどろく」と対応している。
- 例文: 使ひをやりて、おどろかしけり。
- 訳: 使者を送って、（相手を）起こした（気づかせた）。

### id 11 おどろおどろし（形）— 大げさだ、仰々しい／気味が悪い／はなはだしい
- POINT: 「おどろく」から派生した語だが、「驚かせるほど」という程度の大きさを表す方向に意味が広がった。
- 例文: おどろおどろしく泣きののしる。
- 訳: 大げさに泣き騒ぐ。

### id 12–23 「少しも、まったく、決して（打消と呼応）」の副詞群
さらに／つゆ／たえて／すべて／おほかた／つやつや／よに／かまへて／ゆめ（ゆめ）／あへて／かけても／あからさまにも

- POINT（共通）: いずれも下に打消の語（ず・じ・まじ 等）を伴って「少しも〜ない」という**呼応の副詞**。単独では意味が定まらないので、必ず文末の打消とセットで覚える。
- 個別の補足：
  - **ゆめ**：禁止（な〜そ）と結びつくと「決して〜するな」の強い禁止になる。
  - **かまへて**：現代語「かまえて」と同根で「用意して＝念には念を入れて」→打消と結んで「決して」。
  - **をさをさ**（id24、打消の呼応は共通だが意味が異なる＝下記）は「ほとんど〜ない」で程度が違うため区別。

- 例文（さらに）: さらに人の来る音もせず。
- 訳: まったく人が来る音もしない。
- 例文（つゆ）: つゆ知らざりき。
- 訳: 少しも知らなかった。
- 例文（かまへて）: かまへて心を許すことなかれ。
- 訳: 決して心を許すことがあってはならない。

### id 24 をさをさ（副）— ほとんど（打消と呼応）
- POINT: id12〜23の「決して」グループと違い、「ほとんど〜ない」と**程度の弱い否定**になる点に注意（完全否定ではない）。
- 例文: 人影、をさをさ見えず。
- 訳: 人影が、ほとんど見えない。

### id 25 な（〜そ）（副）— 〜してくれるな（「そ」は終助詞）
- POINT: 「な＋連用形（またはサ変・カ変は未然形）＋そ」で禁止を表す呼応表現。「な」だけでは意味をなさない。
- 例文: な忘れそ。
- 訳: 忘れてくれるな。

### id 26 うしろめたし／うしろめたなし（後ろめたし／形）— 気がかりだ、心配だ
- POINT: 「後ろ（見えない部分）」が「目立たし（=はっきりしない）」→見えないところが不安、で「心配だ」。現代語「後ろめたい（罪悪感）」とはニュアンスが異なる。
- 例文: 幼き人を都に置きて、うしろめたし。
- 訳: 幼い人を都に置いてきて、気がかりだ。

### id 27 うしろやすし（後ろ安し／形）— 安心だ、頼もしい
- POINT: id26「うしろめたし」の反対語。「後ろ（背後・将来）」が「安し」で心配がない、と対比で覚える。
- 例文: 頼りになる人あれば、うしろやすし。
- 訳: 頼りになる人がいるので、安心だ。

### id 28 かたち（容貌・形／名）— 容貌、顔立ち／容姿
- POINT: 古語では「形」全般ではなく、人の顔立ち・容姿を指すことが中心的な意味。
- 例文: かたちいと清らなる人なり。
- 訳: 容貌がとても美しい人だ。

### id 29 みめ（見目／名）— 見た目、外見／容貌、顔立ち
- POINT: id28「かたち」とほぼ同義で対で覚えるとよい。「見た目」という語のなりたちそのまま。
- 例文: みめよき人を選びて仕へさす。
- 訳: 容貌のよい人を選んで仕えさせる。

### id 30 をかし（形）— 滑稽だ、おかしい／興味深い、おもしろい／趣がある、風情がある／美しい、かわいらしい、魅力的だ／すばらしい、優れている
- POINT: 「もののあはれ」（しみじみとした情趣）と対で語られる、知的・明るい「趣・美」の代表語。意味が広いので文脈判断が最重要。
- 例文: いとをかしき絵をぞ見つけたる。
- 訳: たいへん趣のある（すばらしい）絵を見つけた。

### id 31 をかしげなり（形動）— 美しい、かわいらしい、魅力的だ／趣がある、風情がある
- POINT: 「をかし」＋「げ（〜らしい様子）」＋「なり」。視覚的な美しさ・かわいらしさを表す用法が中心。
- 例文: をかしげなる童のありく。
- 訳: かわいらしい様子の子どもが歩き回っている。

### id 32 おもしろし（面白し／形）— 楽しい、愉快だ、興味深い／趣がある、風情がある
- POINT: 「面（目の前）」が「白し（明るく開ける）」が原義。景色や音楽など対象が明るく心を晴れやかにする、というニュアンス。
- 例文: おもしろき月の夜に、笛を吹く。
- 訳: 趣のある月の夜に、笛を吹く。

### id 33 なやむ（悩む／動）— （病気などで）苦しむ／思い煩う
- POINT: 現代語「悩む（思案する）」より、古語ではまず「病気で苦しむ」の意味が中心であることに注意。
- 例文: 日ごろなやみて、床に伏す。
- 訳: この数日病気で苦しんで、床に伏している。

### id 34 なやまし（悩まし／形）— （病気などで）気分が悪い／心が苦しい、思い煩う様子だ
- POINT: id33「なやむ」の形容詞形。現代語「悩ましい（魅惑的）」とは意味が異なるので誤読注意。
- 例文: なやましき心地して、物も食はず。
- 訳: 気分が悪い感じがして、食事も取らない。

### id 35 わづらふ（患ふ・煩ふ／動）— （病気などで）苦しむ／思い悩む／〜しかねる（動詞の連用形について）
- POINT: 補助動詞用法「〜わづらふ」＝「〜しようとしてもできない」に注意。単独と補助動詞で訳し分ける。
- 例文: 文をやらむとしてわづらふ。
- 訳: 手紙を送ろうとするが送りかねている。

### id 36 わづらはし（患はし・煩はし／形）— 病状が重い／めんどうだ、やっかいだ
- POINT: id35「わづらふ」の形容詞形。現代語「わずらわしい（めんどう）」の意味は共通するが、「病状が重い」の意味も古文特有。
- 例文: わづらはしき病にかかりぬ。
- 訳: 重い病気にかかってしまった。

### id 37 おくる（後る・遅る／動）— 遅れる、取り残される、劣る／先に死なれる、先立たれる
- POINT: 「先立たれる（＝自分が後に残される＝死別）」の意味が入試頻出。単なる「遅刻」の意味だけで訳さないこと。
- 例文: 親におくれて、一人残りぬ。
- 訳: 親に先立たれて、一人残った。

### id 38 やがて（副）— そのまま／すぐに
- POINT: 現代語「やがて（しばらくして）」と時間の向きが逆で、古語は「間を置かずすぐに」の意味が中心。
- 例文: 文を見て、やがて返事を書く。
- 訳: 手紙を見て、すぐに返事を書く。

### id 39 とりあへず（取り敢へず／副）— すぐに
- POINT: 現代語「とりあえず（一旦・暫定的に）」とは異なり、古語は「余裕なく、間に合わせるひまもなく＝すぐに」の意味。
- 例文: とりあへず走り出でぬ。
- 訳: すぐに走って出て行った。

### id 40 うつくし（美し／形）— かわいい、愛らしい／きれいだ、美しい
- POINT: 古語では小さいもの・幼いものへの「かわいらしさ」が第一義。現代語の「美しい（壮麗）」の意味は後発。
- 例文: うつくしき稚児の笑ふ。
- 訳: かわいらしい幼子が笑う。

### id 41 らうたし（形）— かわいい、愛らしい
- POINT: id40「うつくし」と近いが、「らうたし」は庇護したくなるような「いたわりたい」気持ちを含む点が違う。
- 例文: らうたき子を懐に抱く。
- 訳: 愛らしい子を懐に抱く。

### id 42 めやすし（目安し／形）— 感じがよい、見苦しくない
- POINT: 「目（見る人の目）」に「安し（安らか）」＝見ていて安心できる、が原義。現代語「目安（基準）」との混同に注意。
- 例文: ふるまひめやすき人なり。
- 訳: 振る舞いが感じのよい人だ。

### id 43 おこなふ（行ふ／動）— （何かを）する、執り行う／仏道修行をする、勤行する
- POINT: 主語が僧・信心深い人物のときは「仏道修行をする」の意味を最優先に検討する。
- 例文: 僧、山にこもりておこなふ。
- 訳: 僧が、山にこもって仏道修行をする。

### id 44 おこなひ（行ひ／名）— （何かを）行うこと、行動／仏道修行、勤行
- POINT: id43「おこなふ」の名詞形。文脈が宗教的であれば「勤行」と訳す。
- 例文: おこなひ怠りなく勤む。
- 訳: 勤行を怠りなく行う。

### id 45 あく（飽く／動）— 満足する、満たされる／飽きる、いやになる
- POINT: 「満足する」（プラス）と「飽きていやになる」（マイナス）の両方の意味を持つ点に注意。打消を伴うと「満足しない＝名残惜しい」の意味になりやすい。
- 例文: 見れどもあかぬ桜かな。
- 訳: 見ても見ても満足しない（見飽きない）桜だなあ。

### id 46 あかず（飽かず／連）— 満足しない、名残惜しい／飽きない、いやにならない
- POINT: id45「あく」の打消形が定型化した連語。「もっと見ていたい＝名残惜しい」の訳が頻出。
- 例文: あかず見送りて立てり。
- 訳: 名残惜しく見送って立っている。

### id 47 あかぬわかれ（飽かぬ別れ／連）— 名残惜しい別れ
- POINT: id45・46の複合語。「まだ満足していないのに来てしまう別れ」という決まり文句として覚える。
- 例文: あかぬわかれの涙こぼるる。
- 訳: 名残惜しい別れの涙がこぼれる。

### id 48 しる（知る・領る／動）— 認識する、理解する／（男女が）交際する／（土地を）領有する、治める
- POINT: 「男女が知り合う＝深い仲になる」「土地を知る＝治める」という比喩的な意味の広がりが入試で問われやすい。
- 例文: この国をしる人なり。
- 訳: この国を治めている人だ。

### id 49 つれなし（形）— さりげない、平然としている／薄情だ、冷淡だ／変わりがない／思うに任せない
- POINT: 「連れ（関わり）なし」が原義で、「周囲と関わりがないような顔＝素知らぬ様子」から複数の訳が派生する。相手が誰に対して「つれなし」かで訳し分ける。
- 例文: 人の心変はれど、つれなくて過ぐす。
- 訳: 相手の心が変わっても、平然として過ごす。

### id 50 もてなす（動）— ふるまう、装う／取り計らう、処理する／扱う、世話する、大切に扱う
- POINT: 現代語「もてなす（歓待する）」に近い意味は「大切に扱う」のみ。「ふるまう」「処理する」の意味も広く問われる。
- 例文: 何事もなきさまにもてなす。
- 訳: 何事もないふうにふるまう。

### id 51 すく（好く／動）— 恋愛を好む／風流を好む
- POINT: 単に「好き」ではなく、「恋愛」または「風流・和歌など」への傾倒という限定的な対象を持つ点に注意。
- 例文: 若くしてすく心ありけり。
- 訳: 若くして恋愛（または風流）を好む心があった。

### id 52 すき（好き・数奇・数寄／名）— 恋愛を好むこと／風流を好むこと
- POINT: id51「すく」の名詞形。「数寄」の表記から「風流・芸道を好むこと（数寄者）」の意味を連想すると覚えやすい。
- 例文: すきの心深き人なり。
- 訳: 風流を好む心が深い人だ。

### id 53 すきずきし（好き好きし／形）— 恋に一途だ、好色だ／風流だ
- POINT: id51・52と同系列。「すき」を重ねて強調した形容詞。文脈でプラス（風流）かマイナス（好色）かを判断する。
- 例文: すきずきしき歌を詠む。
- 訳: 風流な（恋に熱心な）歌を詠む。

### id 54 いとほし（形）— 気の毒だ、かわいそうだ／かわいらしい、いじらしい
- POINT: 「気の毒」という同情の気持ちから「かわいい」という愛情の気持ちへと意味が広がった語。現代語「いとおしい」の由来。
- 例文: 幼くていとほしき有様なり。
- 訳: 幼くて気の毒な（いじらしい）様子だ。

### id 55 いたはし（労はし／形）— 気の毒だ、かわいそうだ／心配だ、大切だ
- POINT: id54「いとほし」と近い意味。「労（いたはる＝労わる）」の語源から「大事にしたい」という気持ちが根底にある。
- 例文: 病がちなる身をいたはしと思ふ。
- 訳: 病気がちな体を気の毒だと（大切だと）思う。

### id 56 こころぐるし（心苦し／形）— つらい／気の毒だ、かわいそうだ
- POINT: 「心」が「苦し」＝自分の心が痛む、という意味。相手を気の毒に思う気持ちと、自分がつらい気持ちの両方で使われる。
- 例文: 人の嘆きを見るも、こころぐるし。
- 訳: 人の嘆きを見るのも、つらい（気の毒だ）。

### id 57 すさまじ（凄まじ／形）— 興ざめだ、おもしろくない／殺風景だ、寒々としている／寒い、冷たい、白い
- POINT: 現代語「すさまじい（激しい・すごい）」と正反対に近い、「期待外れで興ざめ」というマイナスの意味が古語の基本。
- 例文: 花の散りたる庭こそ、すさまじけれ。
- 訳: 花が散ってしまった庭は、興ざめなものだ。

### id 58 すごし（凄し／形）— もの寂しい／恐ろしい、気味が悪い／すばらしい、美しい
- POINT: id57「すさまじ」と混同しやすいが別語。マイナス（寂しい・恐ろしい）からプラス（すばらしい）まで幅広く、文脈判断が必須。
- 例文: すごき山道を一人行く。
- 訳: 恐ろしい（もの寂しい）山道を一人で行く。

### id 59 うるさし（形）— わずらわしい、めんどうだ／行き届いている、優れている
- POINT: 現代語「うるさい（騒がしい）」とは違う語。「細部まで気を配りすぎて煩わしい」から「行き届いて優れている」まで両義。
- 例文: うるさく気を配る性なり。
- 訳: （煩わしいほど）行き届いて気を配る性格だ。

### id 60 うるせし（形）— 気が利いている、賢い／巧みだ、優れている
- POINT: id59「うるさし」と語源が近いが、こちらはマイナス評価を持たず「有能・巧み」の意味に絞られる。
- 例文: うるせき者を使ひにやる。
- 訳: 気が利いた者を使者にやる。

### id 61 ののしる（動）— 大声をあげる、騒ぎ立てる、大きな音を立てる／評判になる
- POINT: 現代語「罵る（悪口）」の意味は古語には無い。単に「大声・騒がしさ」、転じて「評判が広まる」の意味になる。

### id 62 なかなか（副）— 中途半端に、なまじっか／かえって、むしろ
- POINT: 現代語「なかなか（かなり）」とは異なり、「中途半端にするくらいなら、いっそ〜」という逆接的なニュアンスを持つ。

### id 64 ねたし（嫉し・妬し／形）— しゃくだ、腹立たしい／残念だ、悔しい／（ねたましいほど）すばらしい
- POINT: マイナスの「悔しい」から、逆説的にプラスの「すばらしい（嫉妬するほど）」まで意味が広がる点に注意。

### id 65 こころづきなし（心付き無し／形）— 気にくわない、心ひかれない
- POINT: 「心」が「付く（引かれる）」ことが「無し」＝心が引かれない、という直訳から意味を導ける。

### id 66 めざまし（目覚まし／形）— 気にくわない／すばらしい
- POINT: 「目が覚めるほど」という程度の強さが、身分不相応で「心外だ」と感じるか、驚くほど「すばらしい」と感じるかで訳が分かれる。
- 例文: 身のほど過ぎてめざましき振る舞ひなり。
- 訳: 身分不相応で気にくわない（心外な）振る舞いだ。

### id 67 ものし（物し／形）— 不快だ、気に障る、おもしろくない
- POINT: 「もの（なんとなく）」＋「し」で、はっきり言葉にしづらい不快感を表す。婉曲的な不満の表現。
- 例文: 心にものしと思へど、言はず。
- 訳: 心の中では不快だと思うが、口には出さない。

### id 68 おとなし（大人し／形）— 大人びている／思慮分別がある、落ち着いている／年配だ
- POINT: 現代語「おとなしい（静か）」とは異なり、古語では「大人＝分別がある年長者らしい」がまず基本の意味。
- 例文: おとなしき人の言なれば、従ふ。
- 訳: 分別のある（年配の）人の言葉なので、従う。

### id 69 いとけなし／いときなし／いはけなし（形）— 幼い、あどけない
- POINT: 3語とも同義の異形。「幼くて世慣れていない」様子を表し、多く子どもの描写に使われる。
- 例文: いとけなき身にて、親を失ふ。
- 訳: 幼い身で、親を失う。

### id 70 おこたる（怠る／動）— 怠ける、休む／病気がよくなる、回復する
- POINT: 「病がおこたる＝病気がゆるむ＝快方に向かう」という意味が古文特有で頻出。単に「なまける」だけで訳さないこと。
- 例文: 日ごろの病、ようようおこたりぬ。
- 訳: このところの病気が、ようやくよくなった。

### id 71 おこたり（怠り／名）— 怠慢／過失、過ち／謝罪
- POINT: id70「おこたる」の名詞形だが、「怠慢→そこから生じる過ち→過ちを詫びること」と意味が連鎖的に広がる。
- 例文: おこたりを申して許されぬ。
- 訳: 謝罪を申し上げて許された。

### id 72 ことわりなり（理なり／形動）— もっともだ、当然だ
- POINT: 「ことわり（道理）」＋「なり」＝道理にかなっている、が原義。「もっともだ」と訳す判断・評価表現。
- 例文: 人の怒るもことわりなり。
- 訳: 人が怒るのももっともだ。

### id 73 ことわる（理る／動）— 判断する、明らかにする／説明する／言い訳する、弁明する
- POINT: id72・74と同根。「道理を明らかにする」から「説明する」「弁明する」まで派生する。
- 例文: 事の子細をことわりて申す。
- 訳: 事情の詳細を説明して申し上げる。

### id 74 ことわり（理／名）— 道理、判断、理由／説明、言い訳、弁明／辞退、謝罪
- POINT: id72「ことわりなり」・id73「ことわる」の名詞形。3語をセットで覚えると効率がよい。
- 例文: ことわりを尽くして述ぶ。
- 訳: 道理を尽くして述べる。

### id 75 ながむ（眺む／動）— もの思いにふける／ぼんやりと眺める
- POINT: id76の「ながむ（詠む）」と同音異義語。掛詞として和歌に頻出するため、識別（文脈・活用）が重要。
- 例文: 端近くながめて、月を見る。
- 訳: 縁の近くでもの思いにふけって、月を見る。

### id 76 ながむ（詠む／動）— 朗詠する、詠む
- POINT: id75と同音異義語。「和歌を詠む」の意味であり、対象語（歌・句など）の有無で判別できることが多い。
- 例文: 歌をながめて人に贈る。
- 訳: 歌を詠んで人に贈る。

### id 77 ながめ（眺め／名）— もの思いにふけること／ぼんやりと眺めること
- POINT: id75「ながむ（眺む）」の名詞形。「長雨（ながあめ）」との掛詞にも使われる有名な語。
- 例文: つれづれのながめに、日を暮らす。
- 訳: 所在ないもの思い（物思いにふけること）で、一日を過ごす。
